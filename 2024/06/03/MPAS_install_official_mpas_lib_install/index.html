Preliminary requirements: the easiest route

Installing the full set of I/O libraries required by MPAS can be tedious. To make this easier, we've prepared a shell script that may be helpful:

Preliminary requirements: the easiest route

https://www2.mmm.ucar.edu/projects/mpas/scripts/mpas_lib_install.sh

Notes:

  • Before running or following the above script, you will need to have downloaded the library sources from https://www2.mmm.ucar.edu/projects/mpas/scripts
  • If you already have an MPI library, skip the MPICH installation
  • After editing paths and compiler names at the top, you may be able to run the script, but in general the script may be best used as a guide
