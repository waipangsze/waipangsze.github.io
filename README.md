1. Jekyll is a static site generator written in Ruby by Tom Preston-Werner. It is distributed under the open source MIT license.
2. Jekyll's global settings are in a YAML file called _config.yml. Minimal Mistakes has helped us set up most of them, users only need to fill in some blanks according to the syntax of YAML.
3. Jekyll will generate the page (permalink) corresponding to the markdown file according to the naming rules of your markdown file under the _posts directory. For example, we can create a file named 2020-05-04-note.md
